Majority Element

Given an array nums of size n, return the majority element.

The majority element is the element that appears more than ⌊n / 2⌋ times. You may assume that the majority element always exists in the array.

Example 1:

Input: nums = [3,2,3]
Output: 3

Example 2:

Input: nums = [2,2,1,1,1,2,2]
Output: 2

Solution Code


class Solution {
public:
   
    int majorityElement(vector& nums) {
        int ans;
        int n = nums.size();
        unordered_map freq;
        for(int i = 0 ; i < n ; i ++){
            freq[nums[i]]++;
        }
        for(auto x: freq){
            if(x.second > (n/2)) ans= x.first;
        }
        return ans;
    }
};

Solution Explanation

The problem requires finding the majority element in an array, which is the element that appears more than ⌊n / 2⌋ times. The solution involves:

  • Counting Frequencies: Using an unordered map to count the frequency of each element in the array.
  • Finding Majority: Iterating through the frequency map to find the element with a count greater than ⌊n / 2⌋.
  • Time Complexity: This approach runs in O(n) time, where n is the number of elements in the array, due to the single pass for counting and another pass for finding the majority element.
